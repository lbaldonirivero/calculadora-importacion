# Calculadora de Importación Argentina
### Rivero Commerce LLC

Herramienta para calcular el costo de importación desde Miami (FOB) hasta Argentina, incluyendo todos los impuestos y aranceles.

**➡️ [Abrir calculadora](https://[tu-usuario].github.io/calculadora-importacion/)**

---

#### Qué calcula
- Flete marítimo Miami → Buenos Aires
- CIF Buenos Aires
- Arancel de importación (DI) por categoría
- Tasa de estadística (TE)
- IVA 21% + IVA adicional 10.5%
- Anticipos Ganancias 6% + IIBB 3%
- Honorarios despachante de aduana
- THC / Gastos portuarios
- Almacenaje en puerto
- Transporte final al comercio (opcional)

#### Catálogo incluido
152 SKUs del inventario Rivero (Jun 15, 2026) — iluminación, cerámicos, gabinetes de cocina, ventiladores, herramientas y más.
